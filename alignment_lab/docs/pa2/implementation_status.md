# PA2 Coding Status Matrix

This document tracks the PA2 coding checklist against the current repository state.

Status legend:

- `Done`: implemented in code and wired into the normal CLI flow.
- `Partial`: implemented in code, but the canonical `pa2_*` artifact still has to be generated by running the final experiment sequence.
- `Missing`: not implemented.

Current headline: all previously identified coding gaps are now implemented in code. The remaining work is execution, not missing plumbing. The canonical evidence should come from the new `pa2_*` configs rather than the older mixed debug artifacts.

Runtime logging note:

- Structured per-experiment metrics live in `artifacts/logs/<experiment>.jsonl`.
- Full terminal output for staged PA2 runs can be captured through [`scripts/run_pa2_pipeline.sh`](../../scripts/run_pa2_pipeline.sh), which writes one log file per command under `artifacts/run_logs/<timestamp>/`.
